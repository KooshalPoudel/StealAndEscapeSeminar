StealAndEscapeSeminar Unreal Engine 4.27.2 C++ Project

Instructions to Open the Project

1.  Clone the repository from GitHub using GitHub Desktop or by running:

    git clone https://github.com/YOUR_USERNAME/StealAndEscapeSeminar.git

2.  Install the required software:

    -   Unreal Engine 4.27.2
    -   Visual Studio 2019
    -   Desktop Development with C++ workload
    -   Windows 10 SDK
    -   .NET Framework 4.6.2 Developer Pack

3.  After cloning the repository, navigate to the project folder.

4.  Right-click on StealAndEscape.uproject and select “Generate Visual
    Studio project files”.

5.  Double-click StealAndEscape.uproject to open the project in Unreal
    Engine 4.27.2. The engine will automatically compile the C++ source
    code.

6.  To open the project in Visual Studio 2019, open the generated
    StealAndEscape.sln file, set the configuration to “Development
    Editor” and platform to “Win64”, then build the solution.

7.  If the project fails to open or compile, delete the following
    folders:

    -   Binaries
    -   Intermediate
    -   .vs

    Then regenerate the Visual Studio project files.

8.  Always open the .uproject file located inside the project directory.
    Do not open the Unreal Engine engine solution.
